## Getting Started
```bash
git clone https://github.com/sebhosting/project-aftan.git
cd project-aftan
```

Install backend & frontend dependencies:
```bash
# Backend
cd backend && npm install

# Frontend
cd ../frontend && npm install
```

Run locally:
```bash
# Backend
npm start

# Frontend
npm start
```
